Drawly
======

Simple collaborative drawing demo using flask-socketio


Installation
------------

::

    virtualenv .env
    source .env/bin/activate
    pip install -r requirements.txt
    npm install
    bower install


Running the app
---------------

::

    python manage.py


Then open http://localhost:8000 in your browser
